- 파이썬 스타일 가이드 (PEP8)를 준수하였는지 확인
- 코드 중복을 최소화하고 범용적으로 사용할 수 있도록 모듈화(함수화) 했는지

- **최소한의 코딩으로 결과값을 도출했습니다.**  
- def counter(fn):  
    count = 0  
    def deco_counter():  
        fn()  
        nonlocal count  
        count += 1  
        print(f"{fn.__name__} 실행횟수 : {count}")  
    return deco_counter
